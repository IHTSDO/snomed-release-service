Packages

All packages defined for a given Product will be built and released/published together. Click on the Package name to navigate to a page which will allow you to configure the manifest, input files and post-conditions for each package.

Execution History

This list shows the most recent 'Executions' or runs of the current build. The traffic light indicates the status of each execution (Before Trigger, Queued or Built). Click on the execution link to navigate to a page which shows the output of that particular run of the build and provides an option to publish the results.

Build Triggers

These buttons allow the build to be executed immediately, or configured for an automatic nightly build.
